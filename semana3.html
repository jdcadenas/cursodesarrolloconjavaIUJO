Colecciones y Arreglos

¿Qué es una Colección?

Es un objeto que agrupa múltiples elementos en una sola unidad. Se usan para almacenar, recuperar y manipular datos agregados.

Arreglos (Arrays)

Son estructuras de tamaño fijo. Una vez creados, no pueden crecer.

Estáticos Indexados (0 a n-1) 
public class MainArreglos {
    public static void main(String[] args) {
        // Declaración y uso de arreglo
        int[] notas = new int[3];
        notas[0] = 15;
        notas[1] = 20;
        notas[2] = 18;

        System.out.println("Nota 1: " + notas[0]);
    }
}

La API de Java

Listas Dinámicas: ArrayList

Interfaz List

Una List es una colección ordenada que permite elementos duplicados.

ArrayList: Es la implementación más usada. Funciona como un arreglo que crece automáticamente.

AcciónMétodo
Añadiradd(E e)
Obtenerget(int index)
Eliminarremove(int index)
import java.util.ArrayList;
import java.util.List;

public class MainListas {
    public static void main(String[] args) {
        List<String> lista = new ArrayList<>();
        lista.add("Java");
        lista.add("Python");
        
        System.out.println("Total: " + lista.size());
        System.out.println("Primero: " + lista.get(0));
    }
}

HashSet: Conjuntos Únicos

Sin Duplicados

Un Set es una colección que no permite elementos repetidos. No tiene un orden garantizado.

Uso común: Almacenar Cédulas o correos electrónicos para evitar registros dobles.

import java.util.HashSet;

public class MainSets {
    public static void main(String[] args) {
        HashSet<String> set = new HashSet<>();
        set.add("A");
        set.add("B");
        set.add("A"); // Se ignora

        System.out.println("Tamaño: " + set.size()); // Imprime 2
    }
}

HashMap: Clave e Índice

Mapas (Key-Value)

Almacena pares de datos. Cada elemento tiene una Llave (Key) y un Valor (Value).

Funciona como un "Diccionario" o una lista con índices personalizados.

  • put(k, v): Inserta o actualiza.
  • get(k): Recupera el valor.
import java.util.HashMap;

public class MainMapas {
    public static void main(String[] args) {
        HashMap<String, Integer> edades = new HashMap<>();
        edades.put("Juan", 20);
        edades.put("Maria", 22);

        System.out.println("Edad de Juan: " + edades.get("Juan"));
    }
}

Iteración sobre Colecciones

Formas de Recorrer

Java ofrece varias formas de leer todos los elementos de una colección:

  1. Bucle For-each: El más moderno y limpio.
  2. Iterator: Útil para eliminar elementos mientras recorres.
¡Recomendado: For-each!
import java.util.*;

public class MainIteracion {
    public static void main(String[] args) {
        List<String> items = Arrays.asList("A", "B", "C");

        // 1. For-each
        for (String s : items) { System.out.println(s); }

        // 2. Iterator
        Iterator<String> it = items.iterator();
        while (it.hasNext()) {
            System.out.println(it.next());
        }
    }
}

Reto Final: Gestión de Inventario

Consigna del Reto

Crear un programa interactivo para gestionar el inventario de una tienda. El programa debe mostrar un menú y realizar acciones basadas en la selección del usuario.

Requisitos:

  • Usar un ArrayList para almacenar los nombres de los productos.
  • Implementar un bucle while para que el menú se muestre repetidamente.
  • Dentro del bucle, usar un switch para manejar las siguientes opciones:
    1. Agregar producto: Pide al usuario un nombre y lo añade a la lista.
    2. Buscar producto: Pide un nombre y confirma si existe en la lista.
    3. Eliminar producto: Pide un nombre y lo elimina de la lista.
    4. Listar productos: Muestra todos los productos en el inventario.
    5. Salir.

Solución: Gestión de Inventario

import java.util.ArrayList;
import java.util.Scanner;

public class GestionInventario {
    public static void main(String[] args) {
        ArrayList<String> inventario = new ArrayList<>();
        Scanner sc = new Scanner(System.in);
        int opcion = 0;

        while (opcion != 5) {
            System.out.println("\n--- INVENTARIO IUJO ---");
            System.out.println("1. Agregar producto");
            System.out.println("2. Buscar producto");
            System.out.println("3. Eliminar producto");
            System.out.println("4. Listar productos");
            System.out.println("5. Salir");
            System.out.print("Opción: ");
            opcion = sc.nextInt();
            sc.nextLine(); // Consumir newline

            switch (opcion) {
                case 1:
                    System.out.print("Nombre del producto a agregar: ");
                    inventario.add(sc.nextLine());
                    System.out.println("¡Producto agregado!");
                    break;
                case 2:
                    System.out.print("Nombre del producto a buscar: ");
                    if (inventario.contains(sc.nextLine())) {
                        System.out.println("El producto SÍ está en el inventario.");
                    } else {
                        System.out.println("El producto NO fue encontrado.");
                    }
                    break;
                case 3:
                    System.out.print("Nombre del producto a eliminar: ");
                    if (inventario.remove(sc.nextLine())) {
                        System.out.println("Producto eliminado.");
                    } else {
                        System.out.println("No se pudo eliminar, producto no encontrado.");
                    }
                    break;
                case 4:
                    System.out.println("\n--- Lista de Productos ---");
                    for (String producto : inventario) {
                        System.out.println("- " + producto);
                    }
                    break;
                case 5:
                    System.out.println("Saliendo del sistema...");
                    break;
                default:
                    System.out.println("Opción no válida.");
            }
        }
    }
}
